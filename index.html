How does it work?

The Yellow Pages Extractor scrapes the Yellow Pages for useful information on companies such as name, address, telephone number, fax, email address and website url. If the email address is not available in the Yellow Pages, the scraper will search for an email address on the website of the company, to make sure all public information is properly gathered.

The interface of the scraper is not only easy to use, but will enable you to export the scraped data in a CSV file with the touch of a button. It is perfect for both the technical and non-technical people alike!
